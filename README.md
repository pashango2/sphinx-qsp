# sphinx-quickstart-plus

This is a utility tool that extends 'sphinx-quickstart'.

## Installation

```
pip install sphinx-quickstart-plus
```

## Use

```
sphinx-quickstart-plus
```

## Features

- Remember latest setting
- More extensions
  - commonmark and recommonmark
  - Read the Docs theme
  - sphinx_fontawesome
  - sphinxcontrib-blockdiag
  - nbsphinx
  - sphinx-autobuild
